Music Explorer

This project involved developing a web application as part of a team that reads API data of popular artists and bands, displaying their locations, concert dates, and relevant information such as album releases and creation dates. Utilizing Go (Golang) for the backend and JavaScript for the frontend, we implemented a search functionality that allows users to easily find specific artists and their upcoming events. The application provides an intuitive interface for music fans to explore and stay updated on their favorite artists, enhancing their concert-going experience.
