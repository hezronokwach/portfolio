Covid-19 Global Vaccine Tracker

This project involved creating a Tableau dashboard that tracks the global availability of COVID-19 vaccines, providing real-time data and updates on vaccine distribution. The dashboard displays critical information such as vaccination rates, locations of vaccination sites, and key statistics related to vaccine administration. By integrating various data sources, I ensured that users could easily access and visualize vaccination trends worldwide. The intuitive interface enhances public awareness and supports informed decision-making regarding COVID-19 vaccination efforts.
